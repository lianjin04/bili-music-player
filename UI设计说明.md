# 🎨 B站收藏夹音乐播放器 - UI 设计说明

## 一、页面总览

```
┌──────────────────────────────────┐
│          登录页 LoginPage          │  ← 首次打开
├──────────────────────────────────┤
│          主页 HomePage             │  ← 收藏夹列表
├──────────────────────────────────┤
│     收藏夹详情 FavoriteDetail      │  ← 视频列表
├──────────────────────────────────┤
│         播放器 PlayerPage          │  ← 全屏播放界面
└──────────────────────────────────┘
         ↕ 底部有 PlayerBar 迷你播放栏
```

## 三、视觉风格

| 项目 | 配置 |
|:----|:------|
| 主题 | **Material 3 (Material You)** |
| 主色 | Blue（可自定义） |
| 圆角 | 卡片12px，封面图6~16px |
| 图标 | Material Icons |
| 字体 | 系统默认（Google Fonts 可后续替换） |

### 配色方案 (初版)

```
主色:    #1976D2 (Blue 700)
背景:    #FFFFFF / #F5F5F5
卡片:    #FFFFFF 带阴影
文字:    #212121 / #757575
失效:    #BDBDBD + 红色标签
强调:    #1976D2 (播放按钮)
```

## 四、页面流转图

```
                ┌───────────┐
                │  登录页   │  ← 首次启动 / 退出登录
                └─────┬─────┘
                      │ Cookie验证通过
                      ▼
                ┌───────────┐
           ┌────│   主页    │◄──── 返回
           │    │ (收藏夹)  │
           │    └─────┬─────┘
           │          │ 点击收藏夹
           │          ▼
           │    ┌───────────┐
           │    │ 收藏夹详情 │◄──── 返回
           │    │ (视频列表) │
           │    └─────┬─────┘
           │          │ 点击播放/播放全部
           │          ▼
           │    ┌───────────┐
           └────│  播放器   │  ─── 点击收起 → 返回上一页
                │ (全屏)    │
                └───────────┘
                      │
                      ▼ 返回后
                ┌──────────────┐
                │ 底部播放栏    │  ← 固定在主页/详情页底部
                │ (迷你控制)   │
                └──────────────┘
```
